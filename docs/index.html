Welcome to the Interactive Resource Hub

This website provides interactive demonstrations, visualizations, and Jupyter notebooks that complement the materials presented in class. Each resource is designed to help you develop a deeper understanding of key concepts in network analysis and graph representation learning.

Navigate through the topics using the sidebar to explore interactive elements including network analysis tools, machine learning visualizations, and applications.

Note

These resources are supplementary to the main course content delivered through Canvas. They are designed to provide hands-on exploration of concepts and can be accessed at any time during the course.

How to Use These Resources

Each interactive element is self-contained and can be accessed independently. Click on any item in the sidebar navigation to open the corresponding demonstration or notebook.

For Jupyter notebooks, you can view the rendered HTML version directly in your browser, or download the source files from the course GitHub repository to run them locally.
